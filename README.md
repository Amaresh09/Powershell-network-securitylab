# Powershell-network-securitylab

## Overview
This repository documents an academic lab focused on using PowerShell for
network and security-related administrative tasks in a controlled environment.

The lab demonstrates:
- File content searching using PowerShell
- Ping sweep for host discovery
- Basic port scanning using Test-NetConnection
- Script execution and policy handling in Windows

## Lab Environment
- Windows 10 (PowerShell – Administrator)
- Ubuntu Linux (target for connectivity testing)
- Isolated virtual lab environment

- `docs/Lab14-Writeup.md` – Detailed lab explanation and observations
- `scripts/` – PowerShell scripts used in the lab
- `images/` – Screenshots of execution and results (optional)
